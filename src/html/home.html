  • Import Tools

    MongoDB Atlas Live Migration

    Use Atlas Live Migration to transfer data from a replica set hosted anywhere to an Atlas cluster.

    mongomirror

    Use mongomirror to migrate data from one MongoDB replica set to another.

    mongoimport

    Use mongoimport to load data from a JSON or csv file into a MongoDB cluster.

    mongorestore

    Use mongorestore to load data from a binary database dump into a MongoDB cluster.
